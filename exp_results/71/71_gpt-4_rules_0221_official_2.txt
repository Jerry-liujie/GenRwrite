1. Use JOIN instead of comma-separated FROM clause: This makes the query more readable and easier to understand. It also allows the database to optimize the query better.
2. Move WHERE clause conditions to ON clause in JOINs when possible: This can improve performance by reducing the amount of data that needs to be processed in the join.
3. Use UNION ALL instead of UNION when duplicates are not a concern: UNION ALL is faster because it does not need to check for duplicates.

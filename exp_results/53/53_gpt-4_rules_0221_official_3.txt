1. Use JOIN instead of WHERE for combining tables: This makes the query more readable and easier to understand. It also can improve performance by allowing the database to better optimize the query.
2. Use BETWEEN for range conditions: This simplifies the query and can improve performance by reducing the number of comparisons the database has to make.
3. Move conditions from CASE statement to WHERE clause: This can improve performance by reducing the amount of computation needed. The database can filter out rows that don't meet the conditions before calculating the aggregate functions.

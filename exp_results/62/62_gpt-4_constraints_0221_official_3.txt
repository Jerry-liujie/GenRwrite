1. Subquery Factoring: This rule can be applied when the same complex calculation or condition is used multiple times in the query, especially in aggregate functions or joins.
2. Use IN clause for subquery: This rule can be applied when we need to filter data based on the result of another query, especially when the result set of the subquery is small.
3. Use table alias: This rule can be applied in any query that involves multiple tables. It is especially useful when the table names are long or when the same table is used multiple times in the query.

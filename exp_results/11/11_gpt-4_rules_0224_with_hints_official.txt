2. Replace UNION ALL with separate subqueries: This can improve performance by allowing the database to optimize each subquery independently.

4. Replace WHERE clause with JOIN statement: This can improve performance by reducing the amount of data that needs to be processed. The join operation can filter out unnecessary data early in the query execution, reducing the amount of data that needs to be processed in later stages.

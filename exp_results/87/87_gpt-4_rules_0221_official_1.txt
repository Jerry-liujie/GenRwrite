1. Use JOIN instead of WHERE for combining tables: This makes the query more readable and easier to understand, and in some cases can also improve performance.
2. Use MINUS instead of EXCEPT: Although they are functionally equivalent, some databases do not support EXCEPT but do support MINUS. Using MINUS can therefore improve portability of the query.
3. Remove DISTINCT when using MINUS/EXCEPT: MINUS/EXCEPT automatically remove duplicates, so using DISTINCT is unnecessary and can slow down the query.

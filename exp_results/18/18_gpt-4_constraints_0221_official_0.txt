1. Push down predicate: This rule can be applied when there are conditions in the WHERE clause that can be moved into the FROM clause to filter data at an earlier stage of the query.
2. Use subquery to filter data: This rule can be applied when the query involves joining multiple tables and there are conditions that can be used to filter data before the join operation.
3. Remove unnecessary join: This rule can be applied when a table is joined in the query but its columns are not used in the SELECT, WHERE, GROUP BY or ORDER BY clauses.

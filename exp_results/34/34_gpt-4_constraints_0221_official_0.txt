1. Use explicit JOIN syntax instead of comma-separated tables in FROM clause: This rule can be applied when multiple tables are being joined in a query.
2. Replace BETWEEN operator with IN operator for discrete values: This rule can be applied when the query is filtering based on a discrete set of values.
3. Use BETWEEN operator for continuous range of values: This rule can be applied when the query is filtering based on a continuous range of values.
4. Remove duplicate values in IN clause: This rule can be applied when the query is filtering based on a list of values and there are duplicates in the list.
5. Use JOIN instead of WHERE for joining tables: This rule can be applied when tables are being joined using conditions in the WHERE clause.

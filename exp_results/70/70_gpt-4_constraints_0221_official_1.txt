1. Use CTEs to break down complex queries: This rule can be applied when the query has multiple complex parts that can be separated into individual queries, especially when these parts are used multiple times within the main query.
2. Move ranking calculation to a separate step: This rule can be applied when the query involves ranking or ordering data based on some condition, and this ranking is used in multiple parts of the query.
3. Use a separate CTE for the rollup operation: This rule can be applied when the query involves a rollup operation, especially when the results of this operation are used in multiple parts of the query.

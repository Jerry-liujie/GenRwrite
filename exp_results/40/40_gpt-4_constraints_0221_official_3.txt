1. Move the filtering conditions to the subquery: This rule can be applied when the query involves multiple joins and there are conditions that can be used to filter the data before the join operations.
2. Remove unnecessary cast functions: This rule can be applied when the data types of the columns involved in the operations are known and compatible, and the cast function is used unnecessarily.
3. Use ANSI standard date literals: This rule can be applied when the query involves date operations and the date values are represented as strings.

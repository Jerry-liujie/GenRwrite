1. Push down predicates: This rule can be applied when there are conditions in the main query that can be used to filter data in the subqueries.
2. Use subqueries to pre-filter data: This rule can be applied when there are conditions that can be used to filter data before it is processed in the main query.
3. Avoid using functions in join conditions: This rule can be applied when the join conditions in the query involve functions that can be replaced with equivalent expressions that do not use functions.

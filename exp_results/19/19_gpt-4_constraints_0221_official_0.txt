1. Use JOIN instead of WHERE for linking tables: This rule can be applied when the query involves multiple tables and there are conditions that link these tables together.
2. Remove unnecessary columns from the SELECT clause: This rule can be applied when the query is selecting more columns than are actually needed for the final result.
3. Use LIMIT to reduce the number of rows returned: This rule can be applied when the query is expected to return a large number of rows but only a small subset of the result is actually needed.

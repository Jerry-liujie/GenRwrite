1. Replace implicit joins with explicit joins: This rule can be applied when the query uses a WHERE clause to join tables instead of the JOIN keyword.
2. Use SELECT 1 in EXISTS subqueries: This rule can be applied when the query uses EXISTS subqueries and the actual data returned by the subquery is not used elsewhere in the query.
3. Move join conditions from WHERE clause to ON clause in JOIN operation: This rule can be applied when the query uses a WHERE clause to specify join conditions instead of the ON keyword in the JOIN operation.

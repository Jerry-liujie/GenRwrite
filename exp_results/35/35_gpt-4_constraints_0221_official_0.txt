1. Use JOIN clause instead of comma-separated tables in FROM clause: This rule can be applied when multiple tables are being used in the query and there are relationships between these tables that can be expressed using JOIN clauses.
2. Use SELECT 1 in EXISTS subquery: This rule can be applied when an EXISTS subquery is used in the query and the actual data returned by the subquery is not used in the main query.
3. Use JOIN clause in subquery instead of comma-separated tables: This rule can be applied when a subquery is used in the main query and multiple tables are being used in the subquery that have relationships that can be expressed using JOIN clauses.

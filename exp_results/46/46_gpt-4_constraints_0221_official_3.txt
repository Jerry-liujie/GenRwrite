1. Replace implicit JOINs with explicit JOINs: This rule can be applied when the query uses implicit JOINs (i.e., listing multiple tables in the FROM clause and defining the relationships in the WHERE clause).
2. Remove duplicate values in the IN clause: This rule can be applied when the IN clause of the query contains duplicate values.
3. Remove unnecessary columns from the GROUP BY clause: This rule can be applied when the GROUP BY clause includes columns that are not part of the SELECT clause or part of an aggregate function.
4. Use direct values instead of calculations in the WHERE clause: This rule can be applied when the WHERE clause contains calculations or functions that can be replaced with direct values.

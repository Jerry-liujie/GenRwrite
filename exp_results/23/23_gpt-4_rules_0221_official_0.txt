1. Remove unnecessary columns from subqueries: This reduces the amount of data that needs to be processed and can improve performance.
2. Replace subqueries with joins where possible: Joins can often be more efficient than subqueries.
3. Combine similar subqueries: This reduces redundancy and can improve performance.
4. Use group by before union all: This can reduce the amount of data that needs to be processed in the union operation.
5. Move calculations to the innermost subquery where possible: This can reduce the amount of data that needs to be processed.

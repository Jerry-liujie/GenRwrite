2. Remove unnecessary parentheses: Simplifies the query and improves readability.
3. Use JOIN instead of WHERE for joining tables: Improves performance by reducing the amount of data that needs to be processed.
4. Move join conditions from WHERE clause to ON clause in JOIN statements: Improves performance by reducing the amount of data that needs to be processed.
5. Use IN instead of multiple OR conditions: Simplifies the query and can improve performance by reducing the number of comparisons that need to be made.

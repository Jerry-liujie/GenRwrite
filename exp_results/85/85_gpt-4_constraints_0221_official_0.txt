1. Use JOIN instead of WHERE for linking tables: This rule can be applied when there are multiple tables being queried and the relationships between them are defined using WHERE conditions.
2. Combine conditions on the same table in the same clause: This rule can be applied when there are multiple conditions in the WHERE clause that apply to the same table.
3. Move conditions to JOIN ON clause where possible: This rule can be applied when there are conditions in the WHERE clause that are used to define the relationship between two tables.
4. Remove redundant conditions: This rule can be applied when there are conditions in the WHERE clause that are repeated or unnecessary.

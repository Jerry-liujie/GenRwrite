1. Use explicit JOIN syntax instead of comma-separated FROM clause: This rule can be applied when multiple tables are being joined in a query, especially when there are complex join conditions.
2. Move conditions from WHERE clause to ON clause in JOINs where possible: This rule can be applied when there are conditions in the WHERE clause that are based on columns from multiple tables that are being joined.
3. Use LIMIT clause to reduce the amount of data returned: This rule can be applied when the query is expected to return a large number of rows, but only a subset of the results is actually needed.

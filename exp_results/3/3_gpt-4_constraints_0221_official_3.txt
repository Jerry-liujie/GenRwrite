1. Use explicit JOIN syntax instead of comma-separated FROM clause: This rule can be applied when the query involves combining rows from two or more tables based on a related column between them.
2. Filter in WHERE clause before JOIN: This rule can be applied when there are conditions that can be used to filter the data before performing the join operation.
3. Use LIMIT clause to restrict the number of rows returned: This rule can be applied when the query is expected to return a large number of rows but only a subset of the result is needed.

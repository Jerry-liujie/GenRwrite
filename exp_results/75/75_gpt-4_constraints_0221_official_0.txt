2. Use IFNULL instead of COALESCE: This rule can be applied when the query involves replacing NULL values and only two values are being compared.
3. Use JOIN instead of comma-separated FROM clause: This rule can be applied when the query involves combining rows from two or more tables based on a related column between them.

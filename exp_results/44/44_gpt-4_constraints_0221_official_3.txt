1. Use CTE (Common Table Expressions) for subqueries: This rule can be applied when the same subquery is used multiple times in the query or when the query has complex subqueries that make it hard to read and understand.
2. Use explicit join syntax instead of comma-separated tables in the FROM clause: This rule can be applied when the query involves joining multiple tables and the join conditions are specified in the WHERE clause.
3. Remove unnecessary subqueries: This rule can be applied when there are subqueries that do not contribute to the final result of the query or when the same result can be achieved with simpler expressions.

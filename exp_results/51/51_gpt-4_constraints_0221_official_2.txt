1. Use JOIN instead of comma-separated tables: This rule can be applied when multiple tables are being queried together and there are common columns that can be used to join the tables.
2. Remove unnecessary subqueries: This rule can be applied when there are subqueries that are not contributing to the final result or can be replaced with a simpler operation.
3. Use COALESCE instead of CASE WHEN for NULL handling: This rule can be applied when there are operations in the query that are checking for NULL values and replacing them with some other value.
4. Remove unnecessary aggregation (sum(sum())): This rule can be applied when there are nested aggregation functions that can be simplified to a single aggregation function.

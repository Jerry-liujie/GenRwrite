1. Use JOIN instead of WHERE for table relationships: This rule can be applied when there are multiple tables involved in the query and they are related to each other through certain fields.
2. Remove ROLLUP function: This rule can be applied when the query uses the ROLLUP function but the additional subtotal and grand total rows generated by this function are not needed in the final result.
3. Filter in subquery: This rule can be applied when the outer query processes a large amount of data from a subquery, but some of this data can be filtered out in the subquery itself based on certain conditions.

1. Use CTEs to avoid multiple subqueries: This rule can be applied when the same subquery is used multiple times in the main query or when the main query is complex and can be broken down into smaller, more manageable parts.
2. Combine count and avg calculations in the same subquery: This rule can be applied when multiple aggregate functions are used on the same column or set of columns in the same table.
3. Remove unnecessary parentheses: This rule can be applied when there are parentheses in the query that do not affect the order of operations or the result of the query.

1. Use WITH clause for subqueries: This rule can be applied when the query contains subqueries that are complex or used multiple times in the main query. 
2. Use JOIN instead of comma-separated tables in FROM clause: This rule can be applied when the query involves multiple tables that need to be combined based on some conditions. 
3. Move conditions from WHERE clause to ON clause in JOIN: This rule can be applied when the query contains conditions in the WHERE clause that are related to the joining of tables.

1. Use JOIN instead of WHERE for linking tables: This rule can be applied when there are multiple tables that need to be combined based on a common column.
2. Pre-calculate average returns in a separate subquery: This rule can be applied when there is a need to calculate averages or other aggregate functions and use them in the main query.
3. Use JOIN for subquery instead of WHERE clause: This rule can be applied when a subquery is used to filter or augment the data in the main query.

1. Use JOIN instead of WHERE for combining tables: This rule can be applied when multiple tables are being combined based on matching values in specified columns.
2. Use COUNT instead of SUM for counting: This rule can be applied when the query is trying to count the number of rows that meet a certain condition.
3. Move conditions to JOIN ON clause: This rule can be applied when there are conditions that involve columns from more than one table, and these conditions are used to filter the data before performing other operations.

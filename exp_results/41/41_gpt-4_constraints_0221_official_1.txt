1. Use EXISTS instead of COUNT(*): This rule can be applied when the query is checking for the existence of certain rows, rather than needing the exact count of those rows.
2. Use IN instead of multiple OR conditions: This rule can be applied when there are multiple conditions that check if a column's value is one of several specific values.
3. Combine similar conditions: This rule can be applied when there are multiple conditions that check the same columns with different values, these conditions can be combined using IN operator.
4. Avoid subqueries in the WHERE clause: This rule can be applied when there are subqueries in the WHERE clause that can be rewritten as joins or exists conditions to improve performance.

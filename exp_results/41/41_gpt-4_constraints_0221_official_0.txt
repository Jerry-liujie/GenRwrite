1. Replace multiple OR conditions with IN clause: This rule can be applied when there are multiple OR conditions in the query that are comparing the same field to different values.
2. Replace COUNT(*) with EXISTS: This rule can be applied when the query is using COUNT(*) to check for the existence of records, but the actual count is not needed.
3. Combine similar conditions: This rule can be applied when there are multiple conditions in the query that are checking the same fields with the same or similar values.

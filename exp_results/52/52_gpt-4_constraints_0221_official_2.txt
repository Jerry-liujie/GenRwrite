1. Use explicit JOIN syntax instead of comma-separated FROM clause: This rule can be applied when multiple tables are being queried together and there are relationships between these tables that need to be specified.
2. Move conditions from WHERE clause to ON clause in JOIN statement: This rule can be applied when there are conditions that filter the data based on columns from multiple tables that are being joined.
3. Use LIMIT clause to reduce the number of rows returned: This rule can be applied when only a certain number of top rows are needed from the result set, especially in cases where the results are ordered and only the top or bottom results are relevant.

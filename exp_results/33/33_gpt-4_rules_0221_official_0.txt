1. Combine multiple subqueries into one: This reduces the number of scans of the same table and simplifies the query.
2. Use left join instead of where clause for joining tables: This can improve performance by reducing the number of rows that need to be scanned.
3. Use case statement to calculate sum: This can simplify the calculation and reduce the number of operations.
4. Move conditions from subqueries to main query: This can reduce the number of operations and improve performance.
5. Combine conditions on the same column with in clause: This can reduce the number of comparisons and improve performance.

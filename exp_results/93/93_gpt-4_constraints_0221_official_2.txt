1. Move conditions from WHERE clause to JOIN clause: This rule can be applied when there are conditions in the WHERE clause that involve columns from both tables in a JOIN operation.
2. Remove subquery in FROM clause: This rule can be applied when there is a subquery in the FROM clause that can be rewritten as a JOIN operation.
3. Use IN clause instead of JOIN for filtering: This rule can be applied when a subquery in a JOIN operation returns a small number of rows and can be rewritten using an IN clause.

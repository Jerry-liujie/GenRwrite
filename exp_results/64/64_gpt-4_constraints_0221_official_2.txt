1. Use JOIN instead of WHERE for joining tables: This rule can be applied when there are multiple tables being used in the query and they are connected through a common column or set of columns.
2. Combine multiple range conditions into one: This rule can be applied when there are multiple conditions in the WHERE clause that are checking for a range of values on the same column.
3. Remove unnecessary joins: This rule can be applied when there are tables being joined in the query that are not being used in the SELECT, WHERE, or GROUP BY clauses.

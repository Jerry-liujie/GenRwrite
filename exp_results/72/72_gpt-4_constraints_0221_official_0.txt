1. Filter data as early as possible: This rule can be applied when there are conditions that can be used to filter the data in the early stages of the query.
2. Avoid using LEFT OUTER JOIN if not necessary: This rule can be applied when the results from the LEFT OUTER JOINed tables are not used anywhere in the SELECT or WHERE clauses.
3. Combine conditions on the same table: This rule can be applied when there are multiple conditions on the same table that can be combined into a single condition.
4. Use subqueries to limit data: This rule can be applied when there are conditions that can be used to filter the data within a subquery, thereby limiting the amount of data that needs to be processed in the later stages of the query.

1. Use JOIN instead of comma-separated tables in FROM clause: This rule can be applied when multiple tables are being queried together and there are relationships between these tables that can be expressed as JOIN conditions.
2. Use UNION ALL instead of UNION: This rule can be applied when combining results from multiple SELECT statements and there is no need to remove duplicate rows.
3. Move WHERE clause conditions to the JOIN conditions: This rule can be applied when there are conditions in the WHERE clause that involve columns from multiple tables that are being joined.

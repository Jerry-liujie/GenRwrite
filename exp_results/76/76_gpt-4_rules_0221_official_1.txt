1. Use JOIN instead of comma-separated tables in FROM clause: This makes the query more readable and easier to understand. It also allows the database to optimize the query more effectively.
2. Use UNION ALL instead of UNION: UNION ALL is faster than UNION because it does not need to check for duplicate rows. This is beneficial when you know that the result sets do not contain duplicates.
3. Move WHERE clause conditions to the JOIN conditions: This can improve performance by reducing the number of rows that need to be processed in the join.

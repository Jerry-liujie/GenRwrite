1. Replace the old join style with the new join style: This rule can be applied when the query uses the old-style comma-separated list of tables in the FROM clause with WHERE clause conditions to specify the join conditions.
2. Use subqueries to break down complex conditions: This rule can be applied when the query has complex conditions that involve multiple tables or when the same condition is repeated multiple times in the query.
3. Use CTEs (Common Table Expressions) for repeated subqueries: This rule can be applied when the same subquery is used multiple times in the query. Instead of repeating the subquery, it can be defined once as a CTE and then referenced in the query.

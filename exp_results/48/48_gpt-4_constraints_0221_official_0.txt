1. Use explicit JOIN syntax instead of comma-separated tables in FROM clause: This rule can be applied when multiple tables are being joined in a query, especially when there are complex join conditions.
2. Move join conditions from WHERE clause to ON clause in JOINs: This rule can be applied when there are conditions in the WHERE clause that are actually join conditions between tables.
3. Remove redundant join conditions: This rule can be applied when there are conditions in the query that are repeated or unnecessary for the desired result.
4. Use IN operator for multiple OR conditions on the same column: This rule can be applied when there are multiple OR conditions in the WHERE clause that apply to the same column.

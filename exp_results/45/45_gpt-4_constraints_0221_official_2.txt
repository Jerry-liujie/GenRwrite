1. Replace implicit joins with explicit joins: This rule can be applied when the query uses implicit joins, which are indicated by conditions in the WHERE clause that compare columns from different tables.
2. Remove subquery and replace with direct condition: This rule can be applied when the query contains subqueries that can be replaced with direct conditions in the WHERE clause.
3. Use column directly in condition instead of function on column: This rule can be applied when the query uses functions on columns in conditions, which can prevent the database engine from using indexes effectively.

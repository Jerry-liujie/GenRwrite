1. Use explicit JOIN syntax instead of comma-separated FROM clause: This makes the query easier to read and understand. It also allows for better optimization by the database engine.
2. Filter as early as possible: This reduces the amount of data that needs to be processed in later stages of the query. In this case, we apply the WHERE clause filters before the GROUP BY clause.
3. Use LIMIT clause: This reduces the amount of data that needs to be returned by the query, improving performance.

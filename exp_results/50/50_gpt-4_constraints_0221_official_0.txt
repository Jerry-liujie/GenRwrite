1. Use JOIN instead of WHERE for combining tables: This rule can be applied when multiple tables are being combined based on matching column values, especially when the query involves three or more tables.
2. Move conditions from WHERE clause to ON clause in JOIN statement: This rule can be applied when there are conditions that filter rows based on the relationship between two tables that are being joined.
3. Replace CASE WHEN with IF: This rule can be applied when there are conditional statements that return different values based on whether a certain condition is met.
